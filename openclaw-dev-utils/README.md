# OpenClaw Dev Utilities

A collection of useful bash scripts for developers working with OpenClaw.

## Installation

Clone the repository and add the `bin` directory to your PATH:

```bash
git clone https://github.com/brucestudios/openclaw-dev-utils.git
export PATH="$PATH:/path/to/openclaw-dev-utils/bin"
```

Or install via npm (if we publish as a package):

```bash
npm install -g openclaw-dev-utils
```

## Utilities

### `oc-status`

Provides a quick overview of OpenClaw system status.

```bash
oc-status
```
