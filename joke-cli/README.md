# Joke CLI

A simple command-line interface tool to fetch and display a random joke from the Official Joke API.

## Installation

You can install the joke-cli package via pip:

```bash
pip install joke-cli
```

## Usage

Run the following command to get a random joke:

```bash
joke
```

Or if you have cloned the repository:

```bash
python joke_cli.py
```

## Example Output

```
Why don't scientists trust atoms anymore?
Because they make up everything!
```

## API

This tool uses the [Official Joke API](https://github.com/15Dkatz/official_joke_api).
