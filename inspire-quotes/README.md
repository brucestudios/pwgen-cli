# Inspire Quotes

A simple Python application that generates inspirational quotes. Perfect for getting a quick dose of motivation or wisdom whenever you need it.

## Features

- Generate random inspirational quotes
- List all available quotes
- Get a specific quote by index
- Easy to use command-line interface
- Extensible quote collection

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/inspire-quotes.git
cd inspire-quotes

# Make the script executable (optional)
chmod +x main.py

# Run the application
python3 main.py
```

## Usage

### Get a random quote
```bash
python3 main.py
```

### List all quotes
```bash
python3 main.py --list
```

### Get a specific quote by index
```bash
python3 main.py 5  # Gets the 5th quote
```

### Show help
```bash
python3 main.py --help
```

## Examples

```bash
$ python3 main.py
Spread love everywhere you go. Let no one ever come to you without leaving happier. - Mother Teresa

$ python3 main.py --list
1. The only way to do great work is to love what you do. - Steve Jobs
2. Innovation distinguishes between a leader and a follower. - Steve Jobs
3. Your time is limited, don't waste it living someone else's life. - Steve Jobs
...

$ python3 main.py 3
Your time is limited, don't waste it living someone else's life. - Steve Jobs
```
