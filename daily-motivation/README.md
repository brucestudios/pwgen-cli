# Daily Motivation CLI

A simple command-line tool that fetches and displays a random motivational quote.

## Features

- Fetches a random quote from the ZenQuotes API
- Displays quote with author
- Lightweight and easy to use

## Installation

```bash
git clone https://github.com/yourusername/daily-motivation.git
cd daily-motivation
pip install -r requirements.txt
```

## Usage

Run the script:

```bash
python main.py
```

Example output:

> "The only way to do great work is to love what you do." — Steve Jobs

## Requirements

- Python 3.6+
- requests library

See `requirements.txt` for details.
