# Markdown Enhancer

A small utility to enhance Markdown processing.

## Features

- Convert Markdown to HTML with additional extensions
- Extract tables from Markdown
- Generate table of contents

## Installation

```bash
pip install markdown-enhancer
```

## Usage

```python
from markdown_enhancer import enhance

markdown_text = "# Hello"
html = enhance(markdown_text)
print(html)
```
