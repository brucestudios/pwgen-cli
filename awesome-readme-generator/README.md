# Awesome README Generator
A simple CLI tool to generate high-quality README.md files from templates.
## Table of Contents
- [Installation](#installation)
- [Usage](#usage)
- [License](#license)
- [Contributing](#contributing)

## Installation
```bash
pip install awesome-readme-generator
```
## Usage
```bash
generate_readme --project-name "My Project" --description "A brief description of my project."
```
