# factorial-utils

A simple, well-tested utility for computing factorials in Python.

## Features

- Fast iterative factorial implementation
- Handles large integers (Python's arbitrary precision)
- Comprehensive test suite
- MIT Licensed
- Ready for reuse

## Installation

```bash
pip install factorial-utils
```

## Usage

```python
from factorial_utils import factorial

print(factorial(5))  # 120
print(factorial(0))  # 1
```

## Running Tests

```bash
pytest
```
