# OpenClaw Skill Manager

A command-line utility for managing OpenClaw agent skills.

## Installation

```bash
pip install openclaw-skill-manager
```

## Usage

```bash
oc-skill --help
```

## Commands

- `install <source>`: Install a skill from a local path or URL
- `list`: List installed skills
- `remove <skill_name>`: Remove an installed skill
- `update [skill_name]`: Update an installed skill (if skill_name is omitted, update all)
