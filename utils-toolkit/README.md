# Utils Toolkit

A collection of simple utility functions for common tasks.

## Features

- File operations (copy, move, delete)
- String manipulation
- Simple data validation

## Installation

```bash
pip install utils-toolkit
```

## Usage

```python
from utils_toolkit.file_ops import copy_file

copy_file('source.txt', 'destination.txt')
```

## Development

To install in development mode:

```bash
pip install -e .
```

## Testing

Run tests with:

```bash
pytest
```
