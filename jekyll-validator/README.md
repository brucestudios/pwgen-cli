# Jekyll Validator

A simple command-line tool to validate Jekyll sites, particularly those using the Chirpy theme.

## Features

- Checks for essential Jekyll files (`_config.yml`, `_posts`, etc.)
- Validates common Chirpy theme requirements
- Provides suggestions for fixing issues

## Installation

```bash
pip install jekyll-validator
```

## Usage

```bash
jekyll-validate /path/to/your/jekyll/site
```

## Development

To install in development mode:

```bash
pip install -e .
```
