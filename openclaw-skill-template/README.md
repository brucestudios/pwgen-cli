# OpenClaw Skill Template

A template for creating OpenClaw agent skills.

## Usage

Clone this repository and rename the `skill-template` folder to your skill name.

Then edit the files inside:

- `SKILL.md`: Documentation and metadata for your skill.
- `references/`: Place any reference files or documentation.
- `scripts/`: Helper scripts for your skill.

## Publishing

After developing your skill, you can publish it to [ClawHub](https://clawhub.com) using the `clawhub` CLI:

```bash
clawhub publish
```

Make sure you have configured your ClawHub credentials.
