# Terminal Timer

A simple cross-platform terminal timer with popup notification.

## Features

- Parses time strings like `10s`, `5m`, `1h`, `1h30m10s`
- Waits for the specified duration
- Shows a popup notification when time is up (using tkinter)

## Usage

```bash
python3 terminal_timer.py 10s
python3 terminal_timer.py 5m
python3 terminal_timer.py 1h
python3 terminal_timer.py 1h30m10s
```

## Requirements

- Python 3.x
- tkinter (usually comes with Python on most platforms)
