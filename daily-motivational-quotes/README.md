# Daily Motivational Quote

A simple command-line tool that prints a random motivational quote.

## Features

- Prints a random motivational quote from a curated list
- No external dependencies (uses only Python standard library)
- Lightweight and fast
- Works on any platform with Python 3

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/daily-motivational-quotes.git
   cd daily-motivational-quotes
   ```

2. Make the script executable (optional):
   ```bash
   chmod +x daily_quote.py
   ```

3. Run the script:
   ```bash
   python3 daily_quote.py
   ```

## Usage

Simply run the script to get a random motivational quote:

```bash
$ python3 daily_quote.py
The only way to do great work is to love what you do. – Steve Jobs
```

You can also make it available globally by adding the script to your PATH or creating an alias.

## Example Output

```
Stay hungry, stay foolish. – Steve Jobs
```
