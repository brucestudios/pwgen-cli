# Terminal Dashboard

A simple command-line tool that displays the current time, weather, and system load average.

## Features

- Shows current date and time
- Displays weather for your location (using [wttr.in](https://wttr.in))
- Shows system load average (uptime)

## Requirements

- Python 3.x
- Internet connection (for weather)

## Usage

```bash
python3 dashboard.py
```

## Example Output

```
==================================================
Terminal Dashboard
==================================================
Time: 2026-05-01 11:30:00
Weather: Shanghai: +22°C Sunny
System Load: Load avg: 0.05, 0.10, 0.15
==================================================
```
