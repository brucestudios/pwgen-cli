# FibUtils

A small utility library for computing Fibonacci numbers efficiently.

## Features

- Fast Fibonacci calculation using memoization
- Supports large n (using Python's big integers)
- Simple API
- Well tested

## Installation

```bash
pip install fibutils
```

## Usage

```python
from fibutils.core import fib

print(fib(10))  # 55
```

## Running Tests

```bash
pytest
```
