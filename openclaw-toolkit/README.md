# OpenClaw Toolkit

A collection of useful command-line utilities for OpenClaw users.

## Features

- Simple greetings
- Random jokes
- Password generation
- Text case conversion
- And more...

## Installation

```bash
pip install openclaw-toolkit
```

## Usage

```bash
oc-tool hello
oc-tool joke
oc-tool passgen --length 16
```

## Development

To install in development mode:

```bash
pip install -e .
```
