# Hello OpenClaw

A simple Python package that provides a friendly greeting function.

## Installation

```bash
pip install hello-openclaw
```

## Usage

```python
from hello import greet

print(greet("World"))  # Output: Hello, World!
```

## Development

To install in development mode:

```bash
pip install -e .
```

Running tests:

```bash
pytest
```
