# pwdgen - A Simple yet Powerful Password Generator

`pwdgen` is a Python script that generates strong, customizable passwords with options to include/exclude different character sets, avoid ambiguous characters, and copy to clipboard.

## Features

- 🔒 **Strong Passwords**: Generates cryptographically secure passwords using `random.SystemUnder the hood`
- 🎛️ **Highly Customizable**: Choose which character sets to include (uppercase, lowercase, digits, special characters)
- 🚫 **Ambiguity Exclusion**: Optionally exclude confusing characters like `l`, `1`, `I`, `O`, `0`
- 📋 **Clipboard Support**: Copy generated password directly to clipboard (requires `pyperclip`)
- 📏 **Variable Length**: Set any password length (minimum 4 characters)
- ✅ **Guaranteed Diversity**: Ensures at least one character from each selected character set

## Installation

No installation required! Just download the `pwdgen.py` file and run it with Python 3.

## Usage

```bash
python pwdgen.py [options]
```

### Options

| Option | Description |
|--------|-------------|
| `-l, --length LENGTH` | Length of the password (default: 16) |
| `--no-upper` | Exclude uppercase letters |
| `--no-lower` | Exclude lowercase letters |
| `--no-digits` | Exclude digits |
| `--no-special` | Exclude special characters |
| `--no-ambiguous` | Exclude ambiguous characters (l, 1, I, O, 0) |
| `-c, --clipboard` | Copy the password to the clipboard (requires pyperclip) |
| `-h, --help` | Show this help message |

### Examples

Generate a 16-character password with all character types:
```bash
python pwdgen.py
```

Generate a 20-character password without ambiguous characters, copied to clipboard:
```bash
python pwdgen.py -l 20 --no-ambiguous -c
```

Generate a PIN-like password (digits only, 6 characters):
```bash
python pwdgen.py -l 6 --no-upper --no-lower --no-special
```

Generate a passphrase-like password (letters only, 24 characters):
```bash
python pwdgen.py -l 24 --no-digits --no-special
```

## Requirements

- Python 3.6+
- `pyperclip` (optional, for clipboard functionality)

Install the optional dependency:

## How It Works

The script builds character sets based on your selections, ensures at least one character from each selected set is included, fills the remaining length with random choices from the combined set, shuffles the result, and returns the password.

## Security Note

This script uses Python's `random.SystemRandom()` (via `random.choice()` which defaults to secure random on most systems) for cryptographically secure random number generation suitable for passwords.
