# Inspirational Quotes CLI

A simple command-line tool to display random inspirational quotes.

## Features

- Displays random inspirational quotes from a curated collection
- Lightweight and fast
- No dependencies required (uses only Python standard library)
- Easy to use and install

## Installation

### Option 1: Clone the repository

```bash
git clone https://github.com/yourusername/inspirational-quotes-cli.git
cd inspirational-quotes-cli
```

### Option 2: Install via pip (if published to PyPI)

```bash
pip install inspirational-quotes-cli
```

## Usage

Run the script directly:

```bash
python quotes.py
```

Or make it executable and run it:

```bash
chmod +x quotes.py
./quotes.py
```

## Examples

```
$ python quotes.py
The only way to do great work is to love what you do. – Steve Jobs

$ python quotes.py
Believe you can and you're halfway there. – Theodore Roosevelt

$ python quotes.js
In the middle of every difficulty lies opportunity. – Albert Einstein
```
