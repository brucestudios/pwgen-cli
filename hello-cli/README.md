# Hello CLI

A simple command-line tool to say hello.

## Installation

```bash
pip install hello-cli
```

## Usage

```bash
hello [name] [options]
```

### Arguments

- `name`: The name to greet (default: "World")

### Options

- `-u, --uppercase`: Output the greeting in uppercase

## Examples

```bash
$ hello
Hello, World!

$ hello Alice
Hello, Alice!

$ hello Bruce --uppercase
HELLO, BRUCE!
```
