# Awesome Scripts Collection

A curated collection of useful scripts for daily tasks, automation, and productivity. This repository contains various scripts in different languages that solve common problems developers and sysadmins encounter.

## Contents

- `python/file_organizer.py` - Automatically organize files in a directory by type
- `bash/system_monitor.sh` - Monitor system resources and send alerts
- `javascript/form_validator.js` - Client-side form validation utility
- `LICENSE` - MIT License

## Usage

Each script includes its own documentation in the file header. Make sure to check the requirements and usage instructions for each script.
