# hello-world

A simple OpenClaw skill that prints "Hello, World!".

## Description

This skill provides a single command that outputs "Hello, World!" as a demonstration of how to create and use an OpenClaw skill.

## Tools

This skill does not require any external tools.

## Usage

After installing, you can use the `hello` command to get a greeting.

## Example

```
hello
```
Output:
```
Hello, World!
```

## Installation

Install via ClawHub:

```bash
clawhub install hello-world
```

Or install locally:

```bash
clawhub install /path/to/hello-world
```
