# QRGen CLI

A simple command-line tool to generate QR codes from text or file content. Output can be saved as an image (PNG, SVG) or displayed as ASCII in the terminal.

## Features

- Generate QR codes from plain text or file input
- Output as PNG, SVG, or ANSI/ASCII art in terminal
- Customizable size, border, and error correction
- Lightweight and dependency-free (uses only standard library for ASCII, optional `segno` for image output)

## Installation

```bash
# Clone the repository
git clone https://github.com/your-username/qrgen-cli.git
cd qrgen-cli

# Make the script executable (optional)
chmod +x qrgen.py

# Optionally install segno for image output support
pip install segno
```

## Usage

Generate QR code from text and display in terminal:

```bash
python qrgen.py "Hello, World!"
```

Generate QR code from file and save as PNG:

```bash
python qrgen.py -f input.txt -o qrcode.png
```

Generate SVG QR code:

```bash
python qrgen.py "https://example.com" -o qrcode.svg
```

Show help:

```bash
python qrgen.py --help
```

## Options

- `-t, --text TEXT`: Text to encode (mutually exclusive with `-f`)
- `-f, --file FILE`: Read input from file
- `-o, --output OUTPUT`: Output file path (if not provided, outputs ASCII to terminal)
- `-s, --size SIZE`: Size of each module in pixels (for image output, default 10)
- `-b, --border BORDER`: Border size (default 4)
- `-e, --error ERROR`: Error correction level (L, M, Q, H, default M)
- `--help`: Show this help message

## Examples

```bash
# Simple terminal output
qrgen.py "Scan me!"

# From file, save as PNG
qrgen.py -f data.txt -o mycode.png

# High error correction, larger size
qrgen.py "https://github.com" -o github.png -s 15 -e H

# SVG output
qrgen.py "https://example.com" -o code.svg
```
